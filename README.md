# Backend Demo
Проверить навыки разработки бэкенда на Node.js, включая работу с базой данных, API, аутентификацией

## Используемые технологии

- Node.js
- Express
- PostgreSQL
- Prisma
- Jest
- Docker
- Docker Compose
- TypeScript
- ESLint

## Запуск проекта

```bash
docker compose up -d
```
